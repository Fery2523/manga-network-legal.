# Manga Social Network (Legal & Free)

Este proyecto busca crear una comunidad legal para lectores y autores independientes de manga.

## 🚀 Objetivos
1. Registro de usuarios mediante Firebase.
2. Biblioteca de contenido 100% legal (Autores indie y enlaces oficiales).
3. Sistema de favoritos y seguimiento de capítulos.

## 📂 Categorías Principales
- Sobrenatural
- Psicológico
- Boys' Love (BL)
- Fantasía / Acción

## 🛠️ Stack Tecnológico (Plan Gratuito)
- **Backend:** Python (Flask)
- **Base de Datos:** Firebase Firestore
- **Hosting:** GitHub Pages / Firebase Hosting
- **Editor:** Project IDX / Replit (Mobile friendly)
